# TP1

## Commencement

--

## Une Personne de la Team :

1. Créez un repo sur GitHub, avec un README
2. éditez le README en ligne
3. invitez les collaborateurs

--

## Toute la Team

- Clonez le repo

```bash
git clone <url-repo-distant>
```

--

Voir le [fonctionnement](#how-it-works) de git avant de poursuivre...

--

## Une Personne de la Team :

1. modifiez le README en ajoutant une description du projet
2. ajoutez le README à l’index (`git add README.md`)
3. faites un commit en local (`git commit -m 'Ajout description'`)
4. (testez de revenir à l’état précédent, puis à l’état actuel avec `git checkout <id-commit>`)
5. poussez sur le serveur distant

--

## Les autres de la Team :

1. récupérez-les avec `git pull`
2. visualisez l’arbre des commits avec `git lg --all`
